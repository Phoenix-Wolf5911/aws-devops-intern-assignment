\## EC2 Setup Steps

1\. Launched Ubuntu 22.04 t2.micro instance on AWS EC2

2\. Created security group allowing SSH (22) and HTTP (80)

\## Nginx Installation

sudo apt update \&\& sudo apt upgrade -y

sudo apt install nginx -y

sudo systemctl status nginx



\## Commands Used

\- df -h (disk usage)

\- free -h (memory usage)

\- ps aux (running processes)

\- sudo systemctl restart nginx
